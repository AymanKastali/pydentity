# pydentity

A Python microservice for managing authentication and identity verification.

## Getting Started

### Prerequisites

- Python 3.14+
- [uv](https://docs.astral.sh/uv/)

### Setup

```bash
uv sync
```

### Run

```bash
uv run python -m pydentity
```

### Test

```bash
uv run pytest
```

### Lint

```bash
uv run ruff check .
```

### Type Check

```bash
uv run mypy .
```

## Docker

```bash
docker compose -f docker/docker-compose.yml up --build
```
